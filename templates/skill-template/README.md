# Skill Template

> 创建新 Skill 的标准模板

---

## 📁 目录结构

```
your-skill-name/
├── SKILL.md              # Skill 描述文件（必需）
├── scripts/              # 脚本文件（可选）
├── assets/               # 资源文件（可选）
└── README.md             # 使用说明（可选）
```

---

## 📝 SKILL.md 模板

```markdown
# Skill Name

## Description
简要描述 Skill 的功能和用途。

## Usage
说明如何使用这个 Skill。

## Dependencies
列出依赖项。

## Configuration
配置说明。
```

---

## 🚀 创建步骤

1. 复制此模板文件夹
2. 重命名为你的 Skill 名称
3. 编辑 `SKILL.md` 填写描述
4. 添加必要的脚本和资源
5. 在 `skills/README.md` 中注册

---

## 📋 命名规范

- 使用小写字母和连字符：`my-skill-name`
- 名称应清晰描述功能
- 避免使用空格和特殊字符
